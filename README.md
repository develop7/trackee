Trackee is a tool which periodically grabs screenshots and stores them, making users' activities tracking
much easier.

# Requirements

## Compiling

[Stack](https://haskellstack.com), GDK v3.x development headers.

## Runtime

GDK v3.x

# High-level goal

Match Upwork Team features
